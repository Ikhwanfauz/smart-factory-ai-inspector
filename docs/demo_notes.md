\# Demo Notes



This file will be used to plan the final project demo video.



\## Demo Flow



1\. Open the Smart Factory AI Inspector dashboard

2\. Upload a sample product or steel surface image

3\. Run defect detection

4\. Display detected defect type and confidence score

5\. Run OCR to extract label or serial number

6\. Store the result in the database

7\. Show inspection history and defect statistics

8\. Explain future improvements



\## Demo Video Target Length



1 to 3 minutes.



\## Key Points to Mention



\- The project is designed as an end-to-end AI/ML engineering system

\- It includes model inference, OCR, database logging, and dashboard analytics

\- The system simulates an industrial quality-control workflow

\- Future improvements include video inspection, predictive maintenance, and RAG-based maintenance assistant
