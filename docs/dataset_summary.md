\# Dataset Summary



\## Dataset Name



NEU-DET Steel Surface Defect Dataset



\## Dataset Purpose



This dataset is used for the first version of the Smart Factory AI Inspector project. It supports object detection for industrial steel surface defects, which matches the project goal of building an AI-assisted manufacturing quality-control system.



\## Dataset Format



The dataset was downloaded in YOLOv8 format.
