\# Dataset



This folder is used to store datasets for the Smart Factory AI Inspector project.



\## First Dataset



The first planned dataset is the NEU steel surface defect dataset / NEU-DET.



The dataset contains steel surface defect images with six defect categories:



\- crazing

\- inclusion

\- patches

\- pitted surface

\- rolled-in scale

\- scratches



This dataset is suitable for the first version of the project because it is related to industrial surface inspection and manufacturing quality control.



\## Folder Usage



data/raw/



This folder stores the original downloaded dataset.



data/processed/



This folder stores the cleaned or converted dataset prepared for model training.



\## Notes



Large dataset files are not uploaded to GitHub. Only dataset instructions and processing scripts will be tracked.
